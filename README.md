# multimodal-experiment

“画像選定作業時における複数生体信号計測のための簡易な実験系の実装”

## arduino

ロータリーエンコーダの制御に使用する Arduino ソースコード

## calibration

眼電図と注視点位置を対応付けるためのキャリブレーションタスク実行スクリプト

## python

本実験のためのスクリプト。これとは別に Emotiv TestBench を起動しておく必要があります。
